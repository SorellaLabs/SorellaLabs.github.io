Retrieve socket information for a host, service, or both. Acts as a thin wrapper around the libc getaddrinfo.

The only portable way to support International Domain Names (UTF8 DNS names) is to manually convert to puny code before calling this function - which can be done using the idna crate. However some libc backends may support this natively, or by using bitflags in the hints argument.

Resolving names from non-UTF8 locales is currently not supported (as the interface uses &str). Raise an issue if this is a concern for you.
