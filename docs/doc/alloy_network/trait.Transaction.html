Expand description

Represents a minimal EVM transaction.

Required Associated Types§

source

type Signature

The signature type for this transaction.

This is usually alloy_primitives::Signature, however, it may be different for future EIP-2718 transaction types, or in other networks. For example, in Optimism, the deposit transaction signature is the unit type ().
