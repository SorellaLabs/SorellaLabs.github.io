Expand description

Defines lexicographic tuple generators.

Malachite provides lex_pairs and lex_pairs_from_single, but you can also define lex_triples, lex_quadruples, and so on, and lex_triples_from_single, lex_quadruples_from_single, and so on, in your program using the code below. The documentation for lex_pairs and lex_pairs_from_single describes these other functions as well.

See usage examples here and here.

use malachite_base::iterators::iterator_cache::IteratorCache;
use malachite_base::lex_tuples;

fn clone_helper<T: Clone>(x: &T, _i: usize) -> T {
    x.clone()
}

lex_tuples!(
