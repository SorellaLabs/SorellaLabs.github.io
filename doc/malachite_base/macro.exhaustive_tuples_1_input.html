Expand description

Defines exhaustive tuple generators that generate tuples from a single iterator.

Malachite provides exhaustive_pairs_from_single and exhaustive_pairs_1_input, but you can also define exhaustive_triples_from_single, exhaustive_quadruples_from_single, and so on, and exhaustive_triples_1_input, exhaustive_quadruples_1_input, and so on, in your program using the code below. The documentation for exhaustive_pairs_from_single and exhaustive_pairs_1_input describes these other functions as well.

See usage examples here and here.
