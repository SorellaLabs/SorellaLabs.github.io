Expand description

Resolve a type into a DynSolEvent.

The ResolveSolEvent trait is implemented by types that can be resolved into Solidity-style event descriptors (i.e. a list of indexed parameters and a tuple of non-indexed parameters).

This trait is implemented for alloy_json_abi::Event.
